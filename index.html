Hva er Git?

Git er et veldig nyttig verktøy og hjelpemiddel for koding som kan offentliggjøre prosjekter og gjennfinne feil mye enklere. det git gjør er å skape et oppbevaringsted i prosjektet ditt for å senere lagre andre versjoner av prosjektet, og slik kan du gå tilbake hvis du har ødelagt prosjektet. Man kan si at git er en slags logg.
